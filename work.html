React Native Chat Mobile App

For this project, I was responsible for using React Native to create a mobile chat app, designing the app to a designer's specifications, and deploying the app for use on the App/Play Store.

Using Expo with React Native, I created a chat app that allows users to create a username, select a background color, send text/image/location messages to the chat room. The backend authentication was created as an AWS serverless function, and the chat's database was hosted on Firestore by Google.

I've also created a detailed README using Markdown in Github for testing and collaborative documentation practice.
