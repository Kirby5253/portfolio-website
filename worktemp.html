ReactJS Meetup Progressive Web App

For this project, I created a meetup app that uses the ReactJS framework. The development process was test-driven using Jest, Cucumber, and Puppeteer.

The development objective was to create a Progressive Web App that allows users to access an API of global meetup events from Meetup.com. The app is capable of displaying local event numbers, default location selection, offline access, and global search/suggestion features. The style was cultivated using my own custom CSS where I experimented with some new, lighter color palettes.

Node.js REST API

For this project, I created a Heroku-hosted REST API using the MERN stack. The API was created for the backend of my ReactJS MyFlix app which accessed a MongoDB that I created with details of some of my favorite Marvel Cinematic Universe movies.

The development objective of the API was to learn the MERN tech stack. Since this was my first experiment with Node and the backend, I was a little nervous, but luckily I was able to get the hang of it quickly. The API utilized user authentication and authorization to register new users or login existing users. The app client allowed users the ability to create, read, update, and delete all data.

React Native Chat Mobile App

For this project, I created a mobile chat app using React Native and Expo. The chat app allows users to create a username and select a background color on the landing screen. The user can then send text, capture/send images, and share current location to the chat room once logged in. The backend authentication was created as an AWS serverless function, and the chat's database was hosted on Firestore by Google.

The development objective of this project was to dive into the world of mobile app building with Expo and React Native. I found my experience with React to be helpful in the beginning while I was learning how React Native worked. After the success of this project, I'm excited and looking forward to the next opportunity to create more using React Native.
